# 🚀 화성 기지 비상 창고 암호 해독기 (Mars Mission - Step 8)

## 📝 미션 개요 및 수행 과제
`emergency_storage_key.zip` 파일의 암호를 풀 수 있는 파이썬 코드를 작성합니다.

### 🎯 기본 요구사항
* **암호 조건:** 특수 문자 없이 숫자와 소문자 알파벳으로 구성된 6자리 문자
* **함수명:** 암호를 푸는 메인 코드는 `unlock_zip()` 이라는 이름의 함수로 작성
* **결과 출력:** 암호를 푸는 과정에서 시작 시간, 반복 횟수, 진행 시간 등을 화면에 출력
* **결과 저장:** 암호를 푸는 데 성공하면 암호를 `password.txt` 파일로 저장
* **파일명:** 암호를 풀 수 있는 전체 코드는 `door_hacking.py`로 저장

### ⚙️ 개발 환경 및 제약 조건
* **환경:** Python 3.x 버전
* **라이브러리 제한:** Python에서 기본 제공되는 명령어 및 내장 라이브러리만 사용 (단, zip 파일을 다루는 부분은 예외 허용이나 본 프로젝트는 순수 내장 `zipfile`만 사용)
* **예외 처리:** 파일을 다루는 부분은 반드시 예외 처리가 되어 있어야 함
* **경고 무결성:** 경고 메시지 없이 모든 코드가 정상적으로 실행되어야 함
* **코딩 컨벤션 (PEP 8 준수):**
  * 문자열 표현 시 작은따옴표(`''`)를 기본으로 사용 (부득이한 경우 `""` 사용)
  * 대입문(`=`) 앞뒤로 공백 부여 (예: `foo = (0,)`)
  * 들여쓰기는 공백 4칸 사용

### 3. 보너스 과제 (빠른 해독 알고리즘 구현)
* **ZipCrypto 1차 필터링 & 멀티프로세싱 최적화:** `[매우 우수]`
  * 단순히 코어 수만 늘린 것이 아니라, `zipfile._ZipDecrypter`를 활용해 12바이트 헤더 중 단 1바이트(`check_byte`)만 빠르게 비교하는 **알고리즘 최적화**를 구현했습니다. 무작정 압축을 풀어보는 기존 방식 대비 99.6%의 오답을 즉시 걸러내어 압도적인 해독 속도를 자랑합니다.
  * 다중 프로세스 간 통신(IPC) 오버헤드를 막기 위해 로컬 변수에 시도 횟수를 캐싱한 후 한 번씩만 업데이트하도록 병목 현상을 완벽히 제거했습니다.
  * 정전이나 강제 종료 시에도 데이터가 보존되는 **원자적 체크포인트 저장(Atomic Write)** 기능을 탑재했습니다.
